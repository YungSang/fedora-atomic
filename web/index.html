The Fedora Atomic Initative is a new basis for operating system development and deployment, using a combination of OSTree, the venerable RPM, and the Fedora package set. With rpm-ostree, multiple RPM package configurations can be assembled into an OSTree repository on a build server, tested, and replicated to client machines. Clients using OSTree have fully atomic upgrades.

You can use rpm-ostree inside your organization's firewall to reliably deploy custom operating systems to a plain bare metal server farm, mobile laptops, or virtual cloud deployments, and have them efficiently upgrade while preserving state.

Installation

Pre-provisioned virtual machines ready for use with QEMU/KVM are generated many times a day. You also have the option to install one of these trees inside your existing filesystem, completely non-destructively.

Running your own repository

All the tools here are Free Software - you are not limited to merely consuming pre-constructed trees or packages. You can use rpm-ostree to make your own custom trees derived from the Fedora package set, or include your own custom packages to create a Remix.
